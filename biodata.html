Penjelasan Tugas 3

Capaian Tugas 3 (Vue.js SPA)

1. Penerapan Sistem Pengorganisasian Kode dengan Vue Component & Template
Penerapan: Aplikasi menggunakan arsitektur Single Page Application (SPA). Struktur proyek diorganisasikan dengan baik menggunakan Vue Components (seperti stock-table, do-tracking, order-form, status-badge, dan app-modal) serta Vue Templates terpisah untuk mempermudah pemeliharaan kode.
2. Penerapan Fungsi Menampilkan Data (Mustaches / Directive v-text & v-html)
Penerapan: Menggunakan sintaks ekspresi kumis (mustaches {{ }}) dan directive v-text untuk menampilkan informasi teks dinamis. Directive v-html secara spesifik digunakan untuk merender data deskripsi berformat HTML seperti catatanHTML yang muncul sebagai tooltip interaktif.
3. Penerapan Fungsi Conditional Rendering (v-if / v-else-if / v-else / v-show)
Penerapan: Menggunakan directive v-if, v-else-if, dan v-else pada status ketersediaan stok untuk menampilkan label status ("Aman", "Menipis", "Kosong") dengan gaya warna yang dinamis. Directive v-show diimplementasikan untuk mengatur visibilitas tab navigasi halaman SPA secara instan.
4. Penerapan Data Binding & Property Pengolah Data (v-bind / v-model & Computed / Methods)
Penerapan: Menggunakan v-bind untuk mengikat atribut HTML secara dinamis dan v-model untuk Two-Way Data Binding pada formulir input. Pengolahan data dioptimalkan dengan computed property untuk filter, pengurutan stok, dan penghitungan total harga pesanan, serta methods untuk logika aksi seperti manipulasi data (CRUD) dan pencarian.
5. Penerapan Fungsi Watcher untuk Reaksi Perubahan Data
Penerapan: Menggunakan watchers untuk memantau perubahan variabel secara reaktif. Contohnya digunakan untuk memantau perubahan pilihan filter "UT-Daerah" yang akan memperbarui filter dependen "Kategori Mata Kuliah", serta memantau perubahan input lainnya untuk sinkronisasi state.
6. Penerapan Pengolahan Data Array Melalui Directive v-for
Penerapan: Menggunakan directive v-for untuk merender list data array secara berulang dan efisien, baik untuk daftar stok bahan ajar (menggunakan index angka / zero-based number index) maupun pengulangan opsi dropdown berbasis key-value.
7. Penerapan Formatting Data Teks Menggunakan Konsep Filter
Penerapan: Menggunakan fitur filters pada Vue.js untuk memformat tampilan data tekstual agar lebih rapi, seperti menambahkan simbol mata uang rupiah (Rp) pada nominal harga serta menambahkan satuan (buah) di belakang jumlah stok dan safety stock.
8. Penerapan Penggunaan Custom Element, Vue Component, dan Property Template
Penerapan: Mererefaktor struktur halaman web menjadi elemen kustom berbasis komponen Vue (misal: <ba-stock-table>, <do-tracking>, dll.) yang memiliki properti template terpisah, menghasilkan struktur kode modular yang mudah digunakan kembali.
